Discontinue Watching

About

This plugin allows users to hide items from their "Continue Watching" list. Hidden items will automatically reappear when playback is resumed.



Features

  • Hide items from Continue Watching via API
  • User-specific denylist management
  • Automatic removal from denylist when playback resumes
  • Scheduled cleanup of old Continue Watching items
  • Scheduled cleanup of deleted items from denylists


Scheduled Tasks

Two scheduled tasks are available to help manage your Continue Watching list:

  • Clean Up Continue Watching: Automatically hides items from Continue Watching that haven't been watched within the configured threshold (default: 3 AM daily)
  • Clean Up Denylist: Removes items from user denylists that no longer exist on the server (default: 4 AM daily)

You can manually trigger these tasks or change their schedules from the Jellyfin dashboard under Scheduled Tasks â†’ DiscontinueWatching.



API Endpoints

The following authenticated API endpoints are available:

  • GET /DiscontinueWatching - Get all denylisted items for the current user
  • POST /DiscontinueWatching/{itemId} - Add item to user's denylist
  • DELETE /DiscontinueWatching/{itemId} - Remove item from user's denylist


Client Integration

This plugin includes client-side JavaScript that automatically adds a "Remove from Continue Watching" button to items in the web interface. The button appears when hovering over continue watching items.

Note: The client-side integration requires the JavaScript Injector plugin to be installed and enabled.
